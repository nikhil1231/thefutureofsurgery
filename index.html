About

Our project is based on 3D Imaging and Virtual Reality using Microsoft HoloLens. It’s a combination of making 3D versions of CT scans being integrated into HoloLens so that surgeons and clinicians are able to have a 3D image view of the CT/MR scans using their HoloLens to deeply understand what’s happening inside a specific organ (which normal CT scans aren’t quite able to project that informatively) and the perform Machine Learning/Deep Learning analysis of the CT scans for further recognition in patterns about different scans.

The second part of the idea is about making the archives of different organs which is at the moment used in Medical Academia being available to medical students and teachers for better practical learning. The students will be able to identify and learn a lot more when they are shown the actual 3D model which is dynamic and animated for functionality, for example say the heart with various arteries and veins going through and how the blood flows. This will again be done using the HoloLens where from an archive of large data, smaller chunks of data which could contain heatmaps or even 2D images will be extracted which after post processing and segmentation would be renderable on the HoloLens

Key Features

  • 3D imaging used for CT/MR scans for both short term illnesses and pains and long-term serious pains and even surgeries (and remote surgeries using HoloLens)
  • Data scanned by the HoloLens is then again studied carefully using the HoloLens and other engines about different patterns in diseases (Tumours for Cancer for example) using different Machine Learning/Deep Learning mechanisms. This analysis is then cross-confirmed with Visual Inspection and patterns predicted based on the different results and connections made between different organs of body. (non-assumptive deep learning)
  • Data scanned by the HoloLens is then again studied carefully using the HoloLens and other engines about different patterns in diseases (Tumours for Cancer for example) using different Machine Learning/Deep Learning mechanisms. This analysis is then cross-confirmed with Visual Inspection and patterns predicted based on the different results and connections made between different organs of body. (non-assumptive deep learning)
  • The clinician will be able to use HoloLens for scans and visualization
  • The data will be analysed using some Cloud machine learning/deep learning service (Azure) for failure prevention
  • Dissection of organs will be an important feature to be implemented for detailed study for both medical and academia reasons. So being able to dissect any organ and explore its inside.
  • Being able to explore the different flows and systems in our body (respiratory, circulatory, etc)
  • Real-time parameters viewing while wearing the HoloLens and real-time data analysis about the patient/organ.

End Goals

  • The clinician is able to get a perfect glance of the organ/illness he/she is looking at using the HoloLens.
  • The patterns recognized by the AI algorithms are able to adopt the patterns after a fixed number of iterations and then being able to detect the different patterns for a specific type of disease/illness.
  • Students studying in the medical field would be better equipped with tools (such as HoloLens) for improving their learning. So, the end goal here is being able to project the theory taught in books at this moment in Virtual Reality so that students, be them Undergrads, Postgrads or PhDs would be able to understand various processes in a better way for their academia.
